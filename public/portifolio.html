Academia Zero Grau

Este projeto foi um desafio incrível onde tive a oportunidade de desenvolver um site completo para a Academia Zero Grau. O objetivo era criar uma plataforma online que refletisse a energia e a dedicação que a academia oferece aos seus clientes no mundo físico. O site foi projetado para ser responsivo, garantindo uma experiência de usuário suave em qualquer dispositivo.

Curse Of Ioher

Curse Of Ioher é um projeto que destaca minha dedicação à acessibilidade na web. Implementei várias funções de acessibilidade, incluindo um modo de alto contraste para usuários com baixa visão, um modificador de fonte para facilitar a leitura e um recurso de Libras para a comunidade surda. Este projeto me permitiu explorar e entender a importância da acessibilidade na web, garantindo que todos os usuários, independentemente de suas habilidades, possam navegar e interagir com o site de maneira eficaz.

JNC Embalagens

O projeto JNC Embalagens foi uma oportunidade de criar um site estático para uma empresa de fabricação de embalagens. O desafio aqui era apresentar a ampla gama de produtos da empresa de uma maneira organizada e atraente. Desenvolvi um design personalizado que reflete a identidade da marca e destaca seus produtos. Além disso, implementei uma galeria de produtos interativa que permite aos visitantes explorar a variedade de embalagens disponíveis. Este projeto me permitiu aprimorar minhas habilidades em design de UI e desenvolvimento front-end.

Cooperative Game

Cooperative Game foi um projeto final do meu ensino médio técnico em multimídia. O desafio era criar um jogo de enigma, e o resultado foi um jogo totalmente online, construído do zero com HTML5, CSS3 e JavaScript Vanilla. Todos os assets foram criados manualmente e exclusivamente para o jogo, em colaboração com meu parceiro de projeto. Este projeto me permitiu aplicar e aprimorar minhas habilidades em design de jogos, desenvolvimento front-end e trabalho em equipe.

RS - Advogados Associados

O projeto RS - Advogados Associados foi uma oportunidade para eu aplicar meus conhecimentos em PHP e MySQL. Desenvolvi tanto o front-end quanto o back-end do site, incluindo a conexão com um banco de dados MySQL. O site possui um portal de administração, um gerenciador de comentários e é protegido contra SQL Injection. Este projeto foi uma grande oportunidade para eu aprimorar minhas habilidades em desenvolvimento full-stack.
